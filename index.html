🧪 天然药物化学 · 典型化合物结构题

给结构式 → 写【俗名 + 二级结构类型 + 临床用途/活性】

考试答题铁律：看结构图 → 报三样 → ①俗名（中文常用名） + ②二级结构类型（如强心苷、黄酮醇） + ③活性/用途（具体如抗疟、强心）。
二级结构判定：一级=大母核（甾体/黄酮/生物碱）；二级=母核往下分（甾体→强心苷；黄酮→黄酮醇）←写到这一级；三级=再细分（强心苷→甲型/乙型）←不用写。
例：地高辛 → 写「强心苷」即可，不写「甲型强心苷」

💊 甾体强心苷专题

毛地黄毒苷 Digitoxin：苷元 C3,C14-OH；3分子洋地黄毒糖；强心代表，脂溶性强

地高辛 Digoxin ★：C12多一个-OH；临床最常用；血浆 t½≈33h；治疗窗窄需监测

去乙酰毛地黄毒苷：毛花苷C去乙酰；注射用，起效较快

黄夹苷甲/乙：黄花夹竹桃；甲型强心苷元

⚡ 甲型 vs 乙型强心苷元

比较项甲型（强心甾烯）乙型（海葱甾二烯）
C17内酯环五元 α,β-不饱和-γ-内酯六元 双烯-δ-内酯
碳原子数C₂₃C₂₄
UV最大吸收~218 nm~295 nm
毒性相对较小乙型 > 甲型
代表洋地黄类、黄夹苷海葱、蟾酥

🧪 鉴别反应速记

  • 2-去氧糖：强心苷特有 → 洋地黄毒糖、加拿大麻糖
  • K-K反应：检查2-去氧糖 → 冰醋酸-FeCl₃/浓硫酸交界显色
  • C17显色(甲型)：Legal/Kedde/Raymond → 五元不饱和内酯阳性
  • 甾核显色：Liebermann-Burchard/氯仿-硫酸 → 甾体母核
  • ★水解扣分点：羟基毛地黄毒苷因C16-OH，强酸水解苷元脱水→三脱水产物
