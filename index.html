给结构 → 写【俗名 + 二级结构类型 + 临床用途/活性】

二级结构判定铁律：一级 = 大母核/总类（甾体、黄酮、生物碱…）→ 二级 = 母核往下分的类型（甾体→强心苷；黄酮→黄酮醇）← 写到这一级；三级 = 再细分（强心苷→甲型/乙型）← 不用写。
例：地高辛 → 写「强心苷」即可，不写「甲型强心苷」

💊 甾体强心苷专题 — 品种速记

毛地黄毒苷 Digitoxin (C₄₁H₆₄O₁₃)：苷元 C3,C14-OH；3 分子洋地黄毒糖；强心代表，脂溶性强、作用持久

地高辛 Digoxin ★：在毛地黄毒苷基础上 C12 多一个 -OH；临床最常用强心苷；血浆 t½≈33h；治疗窗窄需监测中毒

去乙酰毛地黄毒苷 Deslanoside：毛花苷 C 去乙酰；含葡萄糖 + 3 洋地黄毒糖；注射用，起效较快

黄夹苷甲/乙 Thevetin A/B：黄花夹竹桃种子；甲型强心苷元；强心，用于心衰

⚡ 甲型 vs 乙型强心苷元（选择/简答高频考点）

比较项甲型（强心甾烯类）乙型（海葱甾二烯类）
C17 内酯环五元 α,β-不饱和-γ-内酯环六元 双烯-δ-内酯环
碳原子数C₂₃C₂₄
UV 最大吸收~218 nm~295 nm
毒性相对较小乙型 > 甲型
代表洋地黄/毛地黄类、黄夹苷海葱、蟾酥类
显色反应Legal/Raymond/Kedde 阳性一般不呈这些反应

🧪 强心苷鉴别反应速记

  • 2-去氧糖（α-去氧糖）：强心苷特有 → 洋地黄毒糖、加拿大麻糖、夹竹桃糖
  • K-K 反应：检查 2-去氧糖 → 冰醋酸-FeCl₃ / 浓硫酸，两液面交界显色
  • C17 内酯环显色（甲型）：Legal / Kedde / Raymond / Baljet → 检查甲型五元不饱和内酯
  • 甾核显色：Liebermann-Burchard / 氯仿-硫酸 → 检查甾体母核
  • ★ 水解扣分点：羟基毛地黄毒苷因 C16-OH，强酸水解时苷元脱水 → 得"三脱水产物"而非原苷元

📝 答题踩分要点

  • ① 俗名写中文常用名：芦丁、小檗碱、青蒿素；不写化学全名
  • ② 结构类型只写到二级：✓ 强心苷/黄酮醇/倍半萜/异喹啉类生物碱；✗ 甲型强心苷（三级）；✗ 只写"甾体"（一级太粗会扣分）
  • ③ 活性写具体：抗疟/强心/平喘，不写笼统的"有药理作用"
  • ④ 不画结构图：只需看图 → 报【俗名+二级类型+活性】三样
