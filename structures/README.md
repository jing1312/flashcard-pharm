# structures/ 目录说明

此目录用于存放化合物的结构式图片。

## 📌 如何替换为 ChemDraw 结构图

1. 在 ChemDraw 中绘制化合物结构式
2. 导出为 PNG 格式（建议 400×300 或更大，白底黑线）
3. 按下方命名规则保存到本目录

## ⚙️ 替换后自动生效

将图片放入本目录后，闪卡会自动优先使用本地图片（`structures/XX-name.png`）。
如果本地图片不存在，则回退使用 PubChem 在线结构图。

## 💡 ChemDraw 导出建议

- 分辨率：至少 300 DPI
- 背景：白色
- 线条：黑色，粗细适中
- 格式：PNG（透明背景也可以）
- 尺寸：建议宽度 400-600px
