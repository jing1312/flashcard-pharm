## 📝 答题踩分要点

1. 俗名写中文常用名，不写化学全名
2. 结构类型只写到**二级**（如"强心苷"，不写"甲型强心苷"）
3. 活性写具体（如"抗疟/强心/平喘"），不写笼统的"有药理作用"
4. 不画结构图，只需看图→报【俗名+二级类型+活性】三样
