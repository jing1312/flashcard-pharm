## 📝 答题踩分要点

1. **俗名写中文常用名**：芦丁、小檗碱、青蒿素；不写化学全名
2. **结构类型只写到二级**：
   - ✓ 强心苷 / 黄酮醇 / 倍半萜 / 异喹啉类生物碱
   - ✗ 甲型强心苷（三级，不写）
   - ✗ 只写"甾体""生物碱"（一级，太粗会扣分）
3. **活性写具体**：抗疟/强心/平喘，不写笼统的"有药理作用"
4. **不画结构图**：只需看图→报【俗名+二级类型+活性】三样
